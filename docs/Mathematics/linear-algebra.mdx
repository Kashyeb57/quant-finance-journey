# Linear Algebra

Vectors, matrices, eigenvalues, and SVD — used in portfolio variance and PCA.

## Key topics

- **Vectors and matrices** — the language of multi-asset portfolios
- **Matrix operations** — multiplication, inversion, solving $Ax = b$
- **Eigenvalues and eigenvectors** — directions a matrix only stretches
- **Singular Value Decomposition (SVD)** — the master factorization behind PCA

## Why quants care

The single most-used formula in portfolio theory is pure linear algebra — the variance
of a portfolio with weights $w$ and covariance matrix $\Sigma$:

$$
\sigma_p^2 = w^\top \Sigma\, w
$$

Eigen-decomposition of that same $\Sigma$ gives **Principal Component Analysis**: the top
eigenvectors are the market's dominant risk factors (level, slope, curvature in rates;
market/sector factors in equities). See [Matrix Factorization](./matrix-factorization) for
the deep dive.

## Try it — portfolio variance in numpy

<PySandbox title="Portfolio risk is a matrix product" code={`import numpy as np

# Three assets: vols 20%, 15%, 10%, with correlations
vols = np.array([0.20, 0.15, 0.10])
corr = np.array([
    [1.0, 0.6, 0.2],
    [0.6, 1.0, 0.4],
    [0.2, 0.4, 1.0],
])
Sigma = np.outer(vols, vols) * corr   # covariance matrix

w = np.array([0.5, 0.3, 0.2])         # portfolio weights

var_p = w @ Sigma @ w                 # the famous w'Σw
print("portfolio vol:", round(float(np.sqrt(var_p)) * 100, 2), "%")

# Diversification check: weighted-average vol vs portfolio vol
print("naive avg vol:", round(float(w @ vols) * 100, 2), "%")

# Eigenvalues of Σ = variance carried by each principal risk factor
eigvals = np.linalg.eigvalsh(Sigma)[::-1]
print("risk factor shares:", np.round(eigvals / eigvals.sum(), 3))`} />

Notice the portfolio vol comes out *lower* than the weighted average of the individual
vols — that gap **is** diversification, and it lives entirely in the off-diagonal
covariance terms.
