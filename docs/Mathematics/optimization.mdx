# Optimization

Linear, nonlinear, constrained, and convex optimization — Markowitz portfolios and beyond.

## Key topics

- **Linear & nonlinear optimization** — maximize/minimize an objective
- **Constrained optimization** — Lagrange multipliers, KKT conditions
- **Convex optimization** — the "nice" class where local = global optimum
- **Quadratic programming** — quadratic objective + linear constraints

## Why quants care

The founding problem of modern portfolio theory is a **quadratic program**. Markowitz
(1952): among all portfolios hitting a target return, hold the one with minimum variance:

$$
\min_w \; w^\top \Sigma w
\quad \text{s.t.} \quad w^\top \mu = \mu_{\text{target}}, \;\; \textstyle\sum_i w_i = 1
$$

Everything else you'll meet — risk-adjusted performance optimization, index tracking,
option hedging under constraints, ML model training — is the same story with a different
objective function.

## Try it — find the minimum-variance portfolio

<PySandbox title="Two assets: the classic closed form" code={`import numpy as np

# Two assets
vol = np.array([0.20, 0.10])
rho = 0.3
Sigma = np.array([
    [vol[0]**2,            rho*vol[0]*vol[1]],
    [rho*vol[0]*vol[1],    vol[1]**2       ],
])

# Minimum-variance weights (closed form: w ∝ Σ⁻¹·1)
ones = np.ones(2)
w = np.linalg.solve(Sigma, ones)
w = w / w.sum()
print("min-variance weights:", np.round(w, 3))
print("portfolio vol:", round(float(np.sqrt(w @ Sigma @ w)) * 100, 2), "%")

# Brute-force check: scan all weight combinations
grid = np.linspace(0, 1, 101)
vols = [np.sqrt(np.array([a, 1-a]) @ Sigma @ np.array([a, 1-a])) for a in grid]
best = grid[int(np.argmin(vols))]
print("grid search says w1 =", best, "-> matches!")

# Challenge: set rho = -0.5 and watch the optimal mix change.`} />
