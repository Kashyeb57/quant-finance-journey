# Numerical Methods

Root-finding, numerical integration, Monte Carlo, and finite difference methods.

## Key topics

- **Root-finding** — Newton–Raphson, bisection
- **Numerical integration** — quadrature and Monte Carlo methods
- **Finite difference methods** — turning PDEs into solvable grids

## Why quants care

Most models **have no closed-form answer** — numerics is how they get solved anyway:

- **Newton–Raphson** is how *implied volatility* is backed out of option prices, millions
  of times a day: find the σ that makes the Black–Scholes price match the market.
- **Monte Carlo** prices path-dependent derivatives by simulating thousands of futures —
  you've already used it in the [GBM simulator](./stochastic-calculus).
- **Finite differences** solve the Black–Scholes PDE on a grid when payoffs get exotic
  (barriers, American exercise).

## Try it — Newton–Raphson root finding

<PySandbox title="Solve x² = 2 the way pricers solve implied vol" code={`# Newton-Raphson: x_new = x - f(x)/f'(x)
# Solve f(x) = x^2 - 2 = 0  (i.e. compute sqrt(2) by hand)

def f(x):  return x*x - 2
def fp(x): return 2*x

x = 1.0                      # initial guess
for i in range(6):
    x = x - f(x) / fp(x)
    print(f"iter {i+1}: x = {x:.10f}")

print("true sqrt(2) =", 2 ** 0.5)

# Notice the DOUBLING of correct digits each step - quadratic convergence.
# Implied-vol solvers do exactly this with f(σ) = BS_price(σ) − market_price.`} />
