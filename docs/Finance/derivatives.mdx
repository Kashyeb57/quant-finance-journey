# Derivatives

Forwards, futures, options, and swaps — the core derivative instruments and how they are priced and used.

A **derivative** is a contract whose value *derives* from something else (a stock, an index, a rate).
The fastest way to build intuition is to play with payoffs — so play:

## 1. Option payoffs, hands-on

Pick a strategy, drag the strike and premium, and watch how the profit/loss profile changes.
Green is where you make money at expiry, red is where you lose it.

<PayoffDiagram />

A few things to try:

- **Long Call** — slide the premium up. Notice the breakeven moves right: the stock must rally *further* just to cover what you paid.
- **Short Put** — this is how "picking up pennies in front of a steamroller" looks: small capped profit, big downside.
- **Long Straddle** — you don't care about direction, only *movement*. Both tails are green.
- **Bull Call Spread** — selling the higher strike finances the lower one, but caps your upside.

## 2. Pricing: the Black–Scholes lab

Payoff diagrams show value *at expiry*. Before expiry an option is worth more than its intrinsic
value — that extra is **time value**, and Black–Scholes tells you how much:

$$
C = S\,N(d_1) - K e^{-rT} N(d_2)
$$

<BlackScholesLab />

Try these experiments:

1. Slide **T → 0** and watch the blue curve collapse onto the dashed hockey stick — time value melting is *theta decay*.
2. Double **σ** — the option gets more valuable even though the spot never moved. Options are long volatility.
3. Set the spot deep in-the-money and check **Delta** — it approaches 1: the option starts behaving like the stock itself.
