# Machine Learning

Supervised and unsupervised learning, model evaluation, and ML applied to financial data.

## What ML means for a quant

ML in quant finance is not about building chatbots — it's about finding signal in noisy financial data, classifying market regimes, estimating probabilities, and proving that a model generalizes instead of memorizing the past. The workhorse tools are regression, classification, and time-series models, with deep learning increasingly used for sequence data. The hard part is rarely fitting the model; it's evaluation — financial data is non-stationary, and a backtest that ignores that will lie to you.

## Why I care about this personally

ML for finance is where my data science degree meets my quant interests most directly — the same models I study in class become trading-signal classifiers and regime detectors when pointed at market data. It's also the pillar where careless methodology is punished fastest, which makes it the best place to practice honest evaluation.
