# Programming

Python, data structures and algorithms, and the engineering skills a quant needs.

## What programming means for a quant

A quant's ideas are only as good as the code that tests them. Programming is where a hypothesis becomes a backtest, a model becomes a live signal, and a paper formula becomes something you can actually trade. Python is the lingua franca for research — fast to write, rich in numerical libraries — while the systems side (data structures, complexity, concurrency, low latency) is what separates a notebook prototype from production infrastructure that executes in microseconds.
