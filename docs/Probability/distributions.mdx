# Random Variables & Distributions

Discrete and continuous distributions and their properties.

The **normal (Gaussian) distribution** is the workhorse of quantitative finance — returns models,
option pricing, and confidence intervals all lean on it. Its density is

$$
f(x) = \frac{1}{\sigma\sqrt{2\pi}}\, e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

but formulas don't build intuition. Sliders do:

Experiments worth running:

1. **Verify the 68–95–99.7 rule.** With μ=0, σ=1 set a=−1, b=1 → ≈68.3%. Then a=−2, b=2 → ≈95.4%.
2. **Slide μ around** — the whole curve shifts but never changes shape. The mean is pure *location*.
3. **Crank σ up** — the curve flattens and spreads. Probability mass leaks into the tails, which is
   exactly what "more risk" means for a return distribution.
4. **Tail risk:** set a=−8, b=−3 with μ=0, σ=1. That tiny sliver is a "3-sigma event" — rarer than
   0.2%. Markets produce them far more often than the Gaussian says they should (fat tails!).
